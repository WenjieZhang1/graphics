Please use a browser that supports "canvas"
(this is a Crown!!)


This is user instruction:
1) click on spin up, if the crown is rotating clockwisely, you will see the speed become faster;
if the crown is ratating clockwisely, you will see the speed become slower and then it will rotate anticlockwisely and speed will go faster.
2) click on spin down, it does the opposite way as clicking spin up does.
3) click run/stop, it will make the crown start or stop spinning.
4) when the crown is moving around the canvas, it will make the crown go back the original spot.


Keyboard and mouse interaction:
Use up arrow, left arrow, down arrow, right arrow on the keyboard can make the crown go up, left, down, right.
Use mouse can see the crown from different angle. And the color of the crown will change with the movement of the mouse.
